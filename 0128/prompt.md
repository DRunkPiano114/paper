# 论文写作任务

你是一名严谨的中文哲学硕士生，任务是：围绕孟子《告子上》"乃若其情，则可以为善矣"中"可以"的义理解释问题，写出一篇不少于5000字的中文论文正文（非开题报告、非大纲），并严格按照学术规范，在你所生成的论文正文中嵌入、校验、补充真实可靠的中文参考文献引注。请严格遵守以下要求：

---

## 0. 写作目标与篇幅

- **输出内容**：一篇"义理哲学取向"的论文正文（不少于5000字），结构可自然分段但不必使用"第一章/第二章"形式；必须形成完整论证闭环（提出问题 → 作出判断 → 论证与反驳 → 结论）。
- **主题聚焦**：孟子性善论中"可以为善"的关键语义与义理地位，尤其是它是否、以及在何种意义上，能够导向"性本善"的理解。

---

## 1. 作者立场与论证重心（最高优先级 · 硬性）

- 论文必须以**作者（"我"）的义理判断为论证起点与中心**，而非以学界分歧本身为中心。
- 我必须围绕《告子上》"乃若其情，则可以为善矣"，直接给出我对"可以"的义理解读，并在全文中持续为该判断辩护。
- 写作目标不是说明"可以有多种读法"，而是回答：
  > **在义理上，哪一种读法是我必须接受、也无法回避的。**

---

## 2. 对"可以为善"的隐性判断约束（不可违背）

在全文论证过程中，必须持续满足以下义理取向（不得在结论或关键论证中与之相反）：

- "可以为善"不得被解释为关于善之必然性的本体规定，而应被理解为一种以"情"为中介、面向道德实践的可能性或能力结构。
- 不得将"可以为善"作为直接推出"性本善（性之所是即善）"的决定性根据；若讨论该推出路径，须明确指出其义理困难或代价。
- "性善"应被理解为一种为道德实践提供内在正当性根据的规范性命题，而非对人性既成状态的事实描述。
- 必须正面论证：正因为"可以为善"并不排除不善的可能，道德判断、责任与实践才具有意义；任何将"可以"理解为必然性的解释，都需面对由此导致的道德意义削弱问题。

> 以上取向不得作为未经论证的前提直接宣告，而须通过文本分析、义理论证与反驳既有观点逐步确立；但全文最终结论不得与之相悖。

---

## 3. 学界既有观点的使用方式（严格限定）

论文必须涉及学界中两种代表性解读，但其功能与地位须严格限定如下：

### 争议背景（须交代但不作为研究目的）

《告子上》"可以"的理解，主要存在两种路径：

1. 将其理解为必然性/本体规定，并由此趋向"性本善"的强断言；
2. 将其理解为能力或可能性结构，并据此强调工夫、养成与实践。

### 严格要求

- 上述观点仅作为我进行义理论证时所面对、评估与回应的既有立场。
- 本文不以系统比较或追溯学界分歧的成因为目标。
- 写作重点在于：在充分理解既有观点最强论证的前提下，我为何仍必须坚持对"可以"的上述理解，并说明其义理优势。

---

## 4. 写作立场与表达方式（硬性）

- **全文必须采用第一人称学术写作方式**（如"我认为""在我看来""我将论证"）。
- 不允许使用回避立场的中性表述，如"本文尝试""尚待进一步研究"等。
- 论文中必须至少一次完整呈现以下结构：
  > 明确反对意见 → 义理回应 → 再反击或限定其成立条件。
- 允许并鼓励使用明确的否定性判断（如"这一理解在义理上难以成立"）。

---

## 5. 文献引用：真实性与核验（硬性）

- 只能使用中文学术文献（中文期刊论文、中文专著、中文学位论文、中文译著）。
- 所有引文必须用于支撑**我在论文中作出的具体判断或反驳**。
- 每条文献须真实存在，并至少经两处可靠来源交叉核验（CNKI / 万方 / 维普 / 期刊官网 / 出版社官网 / 高校机构库 / 国家哲学社科文献中心等）。
- 若存在真实性疑点：
  - 要么不引用；
  - 要么在备注中标注【可能存在不确定性】并说明原因。
- **严禁捏造文献、编造页码、编造 DOI。**

---

## 6. DOI 与参考文献列表（硬性）

- 参考文献须以结构化列表呈现，并尽可能提供有效 DOI：
  ```
  https://doi.org/xxxxx
  ```
- 无 DOI 或无法确认者，须明确标注并提供可核验条目来源。
- **禁止臆造 DOI。**

---

## 7. 输出格式（必须分两部分）

### （一）带引注的论文正文（≥5000字）

- 由你完整生成正文；
- 自然分段即可，不要求章/节编号；
- 所有核心判断须与引文责任相匹配。

### （二）参考文献结构化列表

| 字段 | 说明 |
|------|------|
| id | 文献编号 |
| 作者 | 作者姓名 |
| 年份 | 发表/出版年份 |
| 题名 | 论文或著作名 |
| 来源类型 | 期刊论文/专著/学位论文/译著 |
| 刊名或出版社 | 刊物名称或出版机构 |
| 卷(期) | 期刊卷期号 |
| 页码或章节 | 具体页码范围 |
| DOI | DOI 链接（或明确说明无/未检索到） |
| 核验来源 | 交叉核验的数据库/网站 |
| 备注 | 如【可能存在不确定性】|

---

## 8. 开始工作前的流程

1. 先用 3–5 句话说明：我将如何确立对"可以为善"的义理判断，并如何使用既有观点作为被回应对象。
2. 随后直接输出：
   - （一）带引注的论文正文
   - （二）结构化参考文献列表
